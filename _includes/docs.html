Web Application testing

Below is a sample feature file for testing loggin in feature of your web app.


@UIAutomationTest
Feature: Log into Web App
    As an admin of the system
    I want to goggin into the system providing my credentials

Scenario: Navigate to SUT
    Given I navigate to SUT
    And I enter value "admin" to the "Username" with the "id" of "username"
    And I enter value "abc123" to the "Password" with the "id" of "password"
    And I click on element "submitButton" with the "id" of "btnLogin"

Mobile Application testing

Below is a sample feature file for testing loggin in feature of your mobile app.


@MobileUIAutomationTest
Feature: Log into Android App
    As an admin of the system
    I want to goggin into the system providing my credentials

Scenario: Logging with correct credentials
    Given I enter value "admin" to the "Username" with the "id" of "example.com.mobile:id/username"
    And I enter value "abc123" to the "Password" with the "id" of "example.com.mobile:id/password"
    And I click on element "submitButton" with the "id" of "example.com.mobile:id/btnLogin"
